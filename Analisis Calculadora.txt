El programa inicia, en la interfaz se visualizacion los botones correspondientes a los 10 numeros decimales y las 6 letras extras del lenguaje Hexadecimal, junto a las 4 operaciones (suma, resta, division, multiplicacion), sobre estos botones aparece el espacio de visualizacion de numero, el cual se actualiza al momento de ser pulsado algun boton, se elige la opcion de que sistema o base desea utilizar en un panel lateral, lo cual actualiza tambien la base numerica visualizada en la calculadora, unicamente estan habilitados los botones del sistema numerico seleccionado, de modo que si selecciona el octal, los botones 8, 9, A, B, C, D, E, F quedan deshabilitados, se digita el numero deseado, y se procede a seleccionar una operacion, la pantalla se limpia a la espera de que sea ingresado el siguiente numero, internamente el programa guarda el numero como decimal y dependiendo del sistema numerico seleccionado al momento de darle al = , sera convertido antes de ser visualizado para el usuario, si despues de darle a la operacion deseada se cambia el sistema numerico el segundo numero se escribe con el sistema deseado, recordando que el primer numero ya esta guardado en sistema decimal internamente para facilitar las operaciones sin necesidad de usar vectores o matrices, luego de darle al igual el segundo numero entra al programa, se convierte a decimal, se realiza la operacion con el primero y posteriormente se convierte nuevamente al ultimo sistema elegido, arrojando el resultado en el sistema seleccionado, al cambiar de sistema al ya tener el resultado, unicamente convierte el valor a decimal y posteriormente a su respectiva base y muestra el resultado ya convertido.
